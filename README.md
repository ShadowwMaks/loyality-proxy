# loyality-proxy

Готовый прокси сервер.

### Запуск сервера
1. Создать .env файл (Есть пример - .env.example)
2. Настроить переменные **PORT** *(порт нашего прокси-сервера)* и **TARGET** *(адрес сервера-цели проксирования)*
3. Запустить сервер: 
+ С помощью *npm* 
```
npm install
npm start
```
+ С помощью *yarn* (рекомендуется)
```
yarn
yarn start
```
